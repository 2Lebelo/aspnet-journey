# Projects

This directory contains all the mini ASP.NET projects in this learning journey.

## Project Organization

Each project should:
- Have its own directory
- Include a README.md with project details
- Be self-contained and runnable
- Document what was learned

## Project Template

When creating a new project, include the following in your project's README:

```markdown
# Project Name

## Description
Brief description of what this project does.

## Learning Goals
- What I wanted to learn
- Concepts explored
- Skills developed

## Technologies Used
- ASP.NET Core version
- Additional libraries/frameworks
- Database (if any)

## Prerequisites
- .NET SDK version
- Other dependencies

## How to Run
1. Navigate to project directory
2. Restore dependencies: `dotnet restore`
3. Run the project: `dotnet run`
4. Access at: `http://localhost:5000` (or specified port)

## What I Learned
Key takeaways and insights from building this project.

## Next Steps
Ideas for improvement or related projects to explore.
```
